# Cálculos Básicos em Python

Repositório com exercícios desenvolvidos para praticar operações matemáticas e lógica de programação utilizando Python.

## Exercícios

### 1. Média Escolar Simples
Programa que calcula a média de notas de química, biologia e física.

### 2. Média Ponderada
Programa que calcula a média ponderada de disciplinas utilizando pesos diferentes.

### 3. Cálculo de Conta de Água
Programa que calcula o valor da conta de água com base no consumo mensal e aplica desconto.

## Assuntos praticados

- Variáveis
- Entrada e saída de dados
- Operações matemáticas
- Média simples
- Média ponderada
- Porcentagem
- Formatação de números
- Lógica de programação
